# Trabajo Final - Módulo 4 (JavaScript)
## Proyecto: Aplicación de consola

### Descripción
Este proyecto es una aplicación de consola desarrollada en JavaScript. Se ejecuta en el navegador y utiliza `prompt()` para solicitar datos al usuario, `alert()` para mostrar mensajes y `console.log()` para visualizar resultados en la consola.

### Funcionalidades principales

- Solicita el nombre del usuario y valida:
  - Si el usuario cancela.
  - Si el usuario deja el campo vacío.

- Calculadora con operaciones básicas:
  - Suma, resta, multiplicación y división.
  - Validación para evitar división por 0.
  - Permite repetir operaciones y guarda un historial.

- Arreglos y ciclos:
  - Recorre un arreglo con `for` y `while`.
  - Filtra números “mayores que” o “menores que” un valor ingresado por el usuario.
  
- Objetos:
  - Crea un objeto `usuario` con propiedades y métodos.
  - Crea objetos `operación` y los guarda en un arreglo (historial).
  - Muestra el historial usando `forEach()` y genera un resumen usando `map()`.

### Tecnologías utilizadas
- HTML
- JavaScript (ES6)
- Consola del navegador (DevTools)

### Cómo ejecutar el proyecto
1. Abrir el archivo `index.html` en el navegador.
2. Abrir la consola del navegador con **F12** o **Ctrl + Shift + I**.
3. Ir a la pestaña **Console**.
4. Recargar la página (F5) y seguir las instrucciones de los `prompt()`.

### Estructura del proyecto
- `index.html`: Archivo principal que carga el script.
- `assets/js/script.js`: Código JavaScript del proyecto.

### Capturas sugeridas (para la entrega)
- Consola mostrando el saludo y validación de nombre.
- Consola mostrando una operación realizada (ej: suma) y el resultado.
- Consola mostrando recorrido del arreglo con `for` y `while`.
- Consola mostrando el historial con `forEach()` y el resumen con `map()`.
